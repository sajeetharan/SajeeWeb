## 5-Minute Narration Script

### Opening (0:00 - 0:45)

"Hello everyone! Today, I'm thrilled to announce that the Azure Cosmos DB Extension for Visual Studio Code is now Generally Available. This is a significant milestone for developers working with Azure Cosmos DB, and I want to share why this extension is a game-changer for your development workflow.

For those who may not be familiar, Azure Cosmos DB is Microsoft's globally distributed, multi-model database service. And now, with this VS Code extension, we're bringing the power of Cosmos DB directly into your favorite code editor."

### The Problem & Solution (0:45 - 1:45)

"Let's talk about a common pain point. Traditionally, when developers work with Cosmos DB, they often find themselves constantly switching between their code editor and the Azure Portal. You're writing code in VS Code, then you need to check your data or run a query, so you switch to the browser, navigate to the portal, find your database, execute your query, then switch back to VS Code. This context switching disrupts your flow and slows down development.

That's exactly why we built this extension. We wanted to eliminate that friction and bring all the essential Cosmos DB operations right into VS Code, where you're already spending most of your time. Now, you can manage your entire Cosmos DB workflow without ever leaving your editor."

### Key Features Deep Dive (1:45 - 3:30)

"Let me walk you through the key features that make this extension so powerful.

First, **browsing and managing resources**. With the Azure Cosmos DB explorer integrated into VS Code's sidebar, you can view all your database accounts, databases, containers, and even individual documents in a tree view. It's intuitive, fast, and gives you complete visibility into your Cosmos DB resources.

Second, the **query editor** with full IntelliSense support. This is huge for productivity. You can write and execute NoSQL queries directly in VS Code with syntax highlighting, auto-completion, and error detection. No more switching to the Data Explorer in the portal. You can test queries, see results instantly, and iterate quickly.

Third, **document management**. You can create, view, edit, and delete documents with rich JSON formatting. The extension provides a beautiful, formatted view of your documents, making it easy to work with complex nested structures. You can make changes inline and save them directly to your database.

Fourth, and this is particularly exciting for the AI-assisted development era, we've integrated **context-aware instructions.md and .prompt files**. This feature is designed specifically for agentic development workflows. The extension automatically provides best practices, schema information, and Cosmos DB-specific guidance to GitHub Copilot and other AI coding assistants. This means when you're writing code that interacts with Cosmos DB, your AI assistant has the full context of your database structure, patterns, and recommended approaches. It's like having a Cosmos DB expert sitting alongside your AI pair programmer.

Fifth, **quick actions and commands**. We've added keyboard shortcuts and command palette integration for common operations. Need to create a new container? Just use the command palette. Want to refresh your connection? There's a shortcut for that. These small conveniences add up to significant time savings.

And finally, **native Azure integration**. The extension seamlessly integrates with your Azure account. You can sign in once, and all your Cosmos DB accounts across all your subscriptions are immediately accessible. No need to manage connection strings or authentication tokens manually."
